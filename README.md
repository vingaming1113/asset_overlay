# asset_overlay
my own sober asset overlay folder

# Installing
 To install this,
 Download the repo (with the code --> download zip button or git clone) 
 go to your sober directory ( ~/.var/app/org.vinegarhq.Sober )
 And open the sober folder,
 Now open the asset_overlay folder and copy all files (except readme.md) from the repository you downloaded to it
 If you already have sober open then close and open it again
